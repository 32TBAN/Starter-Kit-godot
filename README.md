# Godot Starter 2D

## Requisitos

- Godot 4.6+

## Cómo empezar

1. Clona el repositorio.
2. Abre el proyecto.
3. Crea una escena en `scenes/levels`.
4. Establece esa escena como nivel inicial desde `Main`.

## Estructura

- assets → Recursos
- core → Servicios globales
- components → Componentes reutilizables
- scenes → Escenas
- scripts → Lógica específica del juego
- resources → Resources personalizadas
- demo → Escenas de prueba
